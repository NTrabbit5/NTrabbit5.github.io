累积日收益

日收益 是指股票价格相对于前一日的损益

累积回报 累积回报是相对于投资计算的，如果累计回报超过一，你就是在盈利，否则就是亏损

累积日收益的公式为

df[daily_cumulative_return] = ( 1+df[pct_daily_return] ).cumprod()
