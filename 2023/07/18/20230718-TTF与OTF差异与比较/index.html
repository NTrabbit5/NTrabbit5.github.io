20230718-TTF与OTF差异与比较

2023-07-18
字数统计: 1k   |   阅读时长≈ 3 分钟

字距是具有特定属性 如大小,粗细 颜色和样式,的可打印文本
Google Docs、Notepad 或 Microsoft Word 等许多软件都设置了默认字体和字体大小。

不同的字体被用于设计网页和文档的样式,有些字体会影响可读性,具体取决于媒体,而有些字体会影响内容的美感

truetype 字体由二次贝塞尔续签构成,而opentype 字体由三次贝塞尔曲线构成

关键精华

  1. TTF (Truetype字体) 是由apple 和Microsoft 开发的一种字体格式, 银企在各种尺寸和分辨率下的简单性和高效光栅化而被广泛使用
  2. OTF (OpenType字体) 是一种基于TTF 的更高级字体格式,由adobe 和Microsoft 创建,提供附加功能, 例如更好地支持连字, 多种字符集和风格变化
  3. TTF 和OTF 都是流行的字体格式,但otf 提供更高级的功能,推荐用于复杂的排版项目

TTF与OTF

TTF 是一种字体，它使用基于矢量的轮廓来定义每个字符。 多年来，它们已广泛用于数字排版和图形设计。 Otf 是一种数字字体文件格式，可以包含范围广泛的排版功能，包括连字、替代字符等。

对比表

比较参数 TTF 场外交易
质质量 TrueType 字体由 Apple 和 Microsoft 开发 OpenType 字体由 Adobe 和 Microsoft 开发
字体文件大小 TTF 字体文件比 OTF 字体文件大 OTF 字体文件通常比 TTF 字体文件小
声望 TTF比OTF更受欢迎 OTF 似乎不如 TrueType 字体受欢迎
可行性 TTF的制作比OFT简单多了 与TTF相比，OTF的创建更难
依赖 TrueType 字体依赖于字形表 OpenType 字体依赖于字形表和 CCF
支持 TTF 一般支持所有旧浏览器 另一方面，由于其高级功能，OTF 支持几乎所有浏览器，无论是旧的还是新的
适应性 最适合日常工作 OTF 发现它适用于设计领域，因为它让用户可以自由地操纵和玩角色
成本 TTF 通常是成本友好的 由于高级功能和规定，购买 OTF 时价格更高。

什么是TTF

TrueType 技术包括两部分

  1. TrueType 光栅器
  2. TrueType-Schriften

对于 TrueType 字体或具有 TrueType 轮廓的 OpenType 字体，使用 .ttf 扩展名。

PostScript 包装器用于管理TrueType 轮廓。

那时候最常用的字体 windows mac 是TrueType.ttf 允许在网站和排放文件中附加字体文件。

TTF 在支持几乎所有旧浏览器方面表现出色。 对于最终用户来说，这是一种成本友好的字体格式。

TTF 主要在日常任务中找到它的应用。 它无法接受输入和产生 产量 正如编程语言所期望的那样，但它的暗示语言提供了编程语言的其他要求。

什么是OTF

OpenType 是 MS Corporation 的官方标志。 OpenType 字体最常用于重要的计算机平台，因为它们具有广泛的可用性和灵活性。

OpenType 字体很灵活，它们可以快速调整大小而不会降低质量

OTF 支持国际字符集

OpenType 字体较新，因此它有更多高级功能供最终用户使用，OTF由于其高级功能，在购买时对用户来说有点贵

它的应用程序主要用于图形设计师的特定设计任务。OpenType 从 1.3 版开始终止了 Multiple Master 功能。

之间的主要区别TTF 和OTF

  1. TrueType 字体的文件扩展名为 TTF，而 OTF 扩展用于 OpenType 字体。
  2. 在这两者中，Truetype 更受欢迎。
  3. OpenType 优于 TrueType 字体。
  4. TrueType 字体比 OTF 旧。
  5. Apple 引入了 TTF，而 Opentype 则起源于 Adobe 和 Microsoft。
  6. 与 TTF 相比，OpenType 字体更难制作。
